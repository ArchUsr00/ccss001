¿Qué es CCSS?

Abreviado en sus siglas en inglés, Cascading Style Sheets, que significa hojas de estilo en cascada, el CSS es una función que se agrega a HTML que proporciona tanto a los desarrolladores de sitios Web, así como a los usuarios, más control sobre cómo se muestran las páginas. Con CSS, los diseñadores y los usuarios pueden crear hojas de estilo que definen cómo aparecen los diferentes elementos, como los encabezados y los enlaces. Estas hojas de estilo se pueden aplicar a cualquier página Web, y nos permite optimizarla para mejorar su posicionamiento.

Partes de una regla CCSS

Una regla de CCSS esta compuesta por:

    "ejemplo: h1 { color:red };"

  • Selector En el ejemplo es el elemento h1
  • Declaración En el ejemplo es la instrucción { color: red; }
  • Propiedad En el ejemplo es la palabra color
  • Valor En el ejemplo es la palabra red

** Conceptos CCSS **

Los conceptos importantes de CCSS son los siguientes:

  • Selectores

    los selectores se utilizan para delimitar los elementos html. El estilo se puede asignar a nivel del Selector.
    Ejemplo:
    El titulo "Ejercicio CCSS Esta condicionado a nivel de elemento H1. Color Blanco y Fondo Azul

  • Especificidad

    La especificidad en CSS es un grupo de reglas aplicadas a los selectores CSS para determinar qué estilo se aplica a un elemento. Cuanto más específico sea un selector, mayor será su valor en puntos y más probable será que esté presente en el estilo del elemento.
    Ejemplo:
    La clase xlabelextab fuerza que el color a usar en esta linea sea azul

  • Cascada

    Cada tipo de selector recibe puntos que indican su especificidad, y se suman los puntos de todos los selectores que hayas utilizado. para calcular la especificidad total del selector.
    • Etiqueta: 1
    • Clase: 10
    • id: 100
    • inline: 1000
    • important: infinito

    Ejemplo:
    esta linea esta compuesta por el id xlabel azul y la clase xclass marrón. Por cascada aplica la marrón

  • Herencia

    Si un elemento no tiene un valor en cascada para una determinada propiedad, puede heredar uno de un elemento antecesor..
